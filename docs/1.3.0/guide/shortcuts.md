# Keyboard Shortcuts

These shortcuts speed up the most common tasks. Many are also shown next to their
commands in the menus.

## File

| Action | Shortcut |
| --- | --- |
| New Project | <kbd>Ctrl</kbd>+<kbd>N</kbd> |
| Open Project… | <kbd>Ctrl</kbd>+<kbd>O</kbd> |
| Save | <kbd>Ctrl</kbd>+<kbd>S</kbd> |
| Save As… | <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>S</kbd> |
| Import to Library… | <kbd>Ctrl</kbd>+<kbd>I</kbd> |
| Add Track… | <kbd>Ctrl</kbd>+<kbd>T</kbd> |
| Export Mixdown… | <kbd>Ctrl</kbd>+<kbd>M</kbd> |
| Exit | <kbd>Ctrl</kbd>+<kbd>E</kbd> |

## In the Scratch Editor

These apply while the [Scratch Editor](/1.3.0/guide/scratch-editor) is open.

| Action | Shortcut |
| --- | --- |
| Play or pause the backing | <kbd>Space</kbd> |
| Build the backing | <kbd>B</kbd> |
| Arm, cancel, or stop recording | <kbd>R</kbd> |
| Play or stop the scratch pattern | <kbd>P</kbd> |
| Clear the pattern | <kbd>C</kbd> |
| Toggle the crossfader cut | <kbd>Z</kbd> or <kbd>M</kbd> |
| Close the editor | <kbd>Esc</kbd> |

Set the crossfader cut key in **Edit ▸ Preferences… ▸ Effects**.

### Editing a scratch pattern

These apply when the Scratch notation editor or one of its points has focus.

| Action | Shortcut |
| --- | --- |
| Undo / redo a notation edit | <kbd>Ctrl</kbd>+<kbd>Z</kbd> / <kbd>Ctrl</kbd>+<kbd>Y</kbd> |
| Delete the selected editable point | <kbd>Delete</kbd> or <kbd>Backspace</kbd> |
| Clear the point selection | <kbd>D</kbd> |
| Nudge the selected point | <kbd>←</kbd> / <kbd>→</kbd> / <kbd>↑</kbd> / <kbd>↓</kbd> |
| Nudge the selected point further | <kbd>Shift</kbd>+<kbd>←</kbd> / <kbd>→</kbd> / <kbd>↑</kbd> / <kbd>↓</kbd> |
| Add a point | <kbd>Insert</kbd> or <kbd>Enter</kbd> |
| Toggle touch on a selected platter point | <kbd>T</kbd> |

With a point selected, **Insert** or **Enter** adds a point on the same lane.
Without a selection, it adds a platter point at the middle of the pattern.
