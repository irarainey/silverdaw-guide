# Keyboard Shortcuts

These shortcuts speed up the most common tasks. Many are also shown next to their
commands in the menus.

## File

| Action | Shortcut |
| --- | --- |
| New Project | <kbd>Ctrl</kbd>+<kbd>N</kbd> |
| Open Project… | <kbd>Ctrl</kbd>+<kbd>O</kbd> |
| Save | <kbd>Ctrl</kbd>+<kbd>S</kbd> |
| Save As… | <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>S</kbd> |
| Import to Library… | <kbd>Ctrl</kbd>+<kbd>I</kbd> |
| Add Track… | <kbd>Ctrl</kbd>+<kbd>T</kbd> |
| Export Mixdown… | <kbd>Ctrl</kbd>+<kbd>M</kbd> |
| Exit | <kbd>Ctrl</kbd>+<kbd>E</kbd> |

## Edit

| Action | Shortcut |
| --- | --- |
| Undo | <kbd>Ctrl</kbd>+<kbd>Z</kbd> |
| Redo | <kbd>Ctrl</kbd>+<kbd>Y</kbd> |
| Cut | <kbd>Ctrl</kbd>+<kbd>X</kbd> |
| Copy | <kbd>Ctrl</kbd>+<kbd>C</kbd> |
| Paste | <kbd>Ctrl</kbd>+<kbd>V</kbd> |
| Trim Project to Last Clip | <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>T</kbd> |
| Rename project | <kbd>F2</kbd> |

## Clips

| Action | Shortcut |
| --- | --- |
| Split Clip at Playhead | <kbd>S</kbd> |
| Duplicate Clip | <kbd>D</kbd> or <kbd>Ctrl</kbd>+<kbd>D</kbd> |
| Delete Clip | <kbd>Delete</kbd> or <kbd>Backspace</kbd> |
| Lock / Unlock Clip | <kbd>Ctrl</kbd>+<kbd>L</kbd> |
| Deselect clip(s) / track (press again to step from clips to track) | <kbd>Esc</kbd> |
| Move selected clip one beat | <kbd>Shift</kbd>+<kbd>←</kbd> / <kbd>→</kbd> |
| Nudge selected clip finely | <kbd>Shift</kbd>+<kbd>Alt</kbd>+<kbd>←</kbd> / <kbd>→</kbd> |

## Tracks

These act on the currently selected track.

| Action | Shortcut |
| --- | --- |
| Mute selected track | <kbd>Shift</kbd>+<kbd>M</kbd> |
| Solo selected track | <kbd>Shift</kbd>+<kbd>S</kbd> |

## Playback and navigation

| Action | Shortcut |
| --- | --- |
| Play / Stop | <kbd>Space</kbd> |
| Toggle the metronome | <kbd>K</kbd> |
| Move playhead one grid line | <kbd>←</kbd> / <kbd>→</kbd> |
| Move playhead finely | <kbd>Alt</kbd>+<kbd>←</kbd> / <kbd>→</kbd> |
| Toggle a marker at the playhead | <kbd>M</kbd> |
| Previous / next marker | <kbd>Ctrl</kbd>+<kbd>←</kbd> / <kbd>→</kbd> |
| Skip to project start / end | <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>←</kbd> / <kbd>→</kbd> or <kbd>Home</kbd> / <kbd>End</kbd> |

## View

| Action | Shortcut |
| --- | --- |
| Zoom In | <kbd>Ctrl</kbd>+<kbd>+</kbd> |
| Zoom Out | <kbd>Ctrl</kbd>+<kbd>-</kbd> |
| Reset Zoom | <kbd>Ctrl</kbd>+<kbd>0</kbd> |
| Zoom to Fit | <kbd>Ctrl</kbd>+<kbd>F</kbd> |
| Jump to 100%–800% zoom | <kbd>Ctrl</kbd>+<kbd>1</kbd> … <kbd>Ctrl</kbd>+<kbd>8</kbd> |
| Toggle Library / FX Panel | <kbd>Ctrl</kbd>+<kbd>J</kbd> |
| Toggle Full Screen | <kbd>F11</kbd> |

## In the Clip Editor

These apply while the [Clip Editor](/1.1.0/guide/clip-editor) is open.

| Action | Shortcut |
| --- | --- |
| Play / Pause | <kbd>Space</kbd> |
| Previous / next beat | <kbd>←</kbd> / <kbd>→</kbd> |
| Nudge playhead finely | <kbd>Alt</kbd>+<kbd>←</kbd> / <kbd>→</kbd> |
| Jump to start / end of playback range | <kbd>Home</kbd> / <kbd>End</kbd> |
| Extend the selection | <kbd>Shift</kbd>+<kbd>←</kbd> / <kbd>→</kbd> |
| Toggle loop playback | <kbd>L</kbd> |
| Toggle the metronome | <kbd>K</kbd> |
| Silence / Full on the selection | <kbd>S</kbd> / <kbd>F</kbd> |
| Zoom in / out / reset | <kbd>+</kbd> / <kbd>-</kbd> / <kbd>0</kbd> |
| Zoom to fit | <kbd>Ctrl</kbd>+<kbd>F</kbd> |
| Clear selection (press again to close) | <kbd>Esc</kbd> |
