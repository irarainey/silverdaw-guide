# Feedback & Support

Silverdaw is built for the people who use it, and your input helps shape it. If
you have run into a problem, want to report a bug, or have an idea that would
make the app better, we would love to hear from you.

## Get in touch

Share feedback, request features, or ask for help through the Silverdaw feedback
portal:

- [silverdaw.featurebase.app](https://silverdaw.featurebase.app)

You can also open it from inside Silverdaw with **Help ▸ Submit Feedback**, which
takes you straight to the portal in your browser.

You can post a new request, browse what other users have suggested, and follow
along as ideas move from suggestion to reality.

## Sending diagnostic logs

If we ask for logs to help track down a problem, Silverdaw can gather them for
you:

1. Turn on **Write diagnostic logs** in **Edit ▸ Preferences ▸ Developer**, then
   restart Silverdaw so logging takes effect.
2. Reproduce the problem so it's captured in the logs.
3. Choose **Help ▸ Send Diagnostic Logs…**. This item appears only while
   diagnostic logging is on.

Silverdaw zips that session's logs into your **Logs** folder, opens the folder so
you can find the zip, and starts a pre-addressed email to
`support@silverdaw.com`. Attach the zip to the email and send it — an email draft
can't attach the file for you, so you add it yourself.

::: tip
The logs are scrubbed of personal details before they're written: your Windows
user name is replaced with `<user>`, and your computer name is never recorded.
:::
